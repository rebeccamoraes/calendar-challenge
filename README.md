# MyCalendar
A calendar application for saving reminders.

## Techs
ReacjJS, TypeScript, Styled Components

## Prerequisites
- Node
- yarn or npm

## Installing and Running
1. git clone https://github.com/rebeccamoraes/calendar-challenge.git
2. yarn
3. yarn start
